Для того, чтобы запустить надо скачать python 3.10.9 https://www.python.org/downloads/release/python-3109/
Ищем ссылку на странице "Windows installer (64-bit)" для Windows
ОБЯЗАТЕЛЬНО УСТАНОВИТЬ ГАЛОЧКУ ПРИ УСТАНОВКЕ "Add to PATH"
Ищем ссылку на странице ""macOS 64-bit universal2 installer" для MacOS

Для проверки установки питона прописать строчку "py --version" 

После установки питона запускаем командную строку в папке со скриптом.
прописываем строчку "pip install -r requirements.txt" (идет скачивание библиотек для скрипта)

Запуск скрипта "py script.py"

Для выбора карты введите цифру
Для выбора команды введите цифру

Далее необходимо указать период
Год(пример:2022)
Месяц(Первые девять месяцев начинают ввод с цифры "0": март - 03, декарь - 12)
День(Первые девять дней начинают ввод с цифры "0": 3 марта - 03, 12 декаря - 12)

Пример ввода

Выберите карту: 4
Введите команду: 1
Введите год от: 2022
Введите месяц от: 12
Введите день от: 11
Введите год до: 2023
Введите месяц до: 03
Введите день до: 11
